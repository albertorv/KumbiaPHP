Framework Kumbia

Es un framework para aplicaciones web libre, escrito en PHP5.

Aporte del framework

    KumbiaPHP intenta proporcionar facilidades para construir aplicaciones robustas para entornos comerciales. Esto significa que el framework es muy flexible y configurable.

Instalación


Paso1: Descargar su archivo comprimido desde https://github.com/KumbiaPHP/KumbiaPHP/


Paso2: Configurar Servidor Web

  1. #a2enmod rewrite
  2. Editar el archivo con direccion etc/apache2/default-server.conf
  3. AllowOverride None debemos de cambiarlo por AllowOverride ALL
  4. Se reinicia el servicio
    
						/etc/init.d/apache2 restart

Ahora podemos crear un nuevo proyecto



						activator new example play-java
					

						activator new example play-scala

Revisamos que que todo este bien



						activator -help

Correr nuestro nuevo proyecto



						activator run
					

						activator ui
					

						activator
